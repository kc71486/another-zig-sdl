### sdl-zig

Yet another sdl3 zig wrapper.
Targets SDL3.2.28

### Warning

This is only intended for personal use, expect bugs and missing features.
If anyone wants a proper wrapper, please use [Gota7/zig-sdl3](https://codeberg.org/7Games/zig-sdl3).

### Get started

Add depencency to `build.zig.zon`.

### Main function

Use `sdl.main.normal()` or `sdl.main.callbacks()` to wrap main, For more info, see source.

### Features

* Provides SDL3 libraries.
